1. Создать папку Alisa_Lessons
2. В папке Alisa_Lessons создать папку 01_Shrimp_class

из github -> https://github.com/yaalisazhukova
-> PyGameLessons -> How -> how PyGame.txt
- поместить его в папку 01_Shrimp_class
4. Создать файл main.py в папке 01_Shrimp_class и добавить в него код из github
how PyGame.txt -> 10
5. В папке 01_Shrimp_class создать папку img и поместить туда
имиджи shrimp.png и cake.png
6. Создать файл sprite.py в папке 01_Shrimp_class
7. В файле sprite.py создать класс Shrimp
8. добавить метод draw и update в класс Shrimp
9. создать экземпляр класса Shrimp в файле main.py
10. отрисовывать картинку Shrimp в цикле while
11. Вызвать метод update() в экземпляре класса Shrimp
